Data Request

Account & Data Deletion

You may request permanent deletion of your ML Toolkit account and all data linked to your ML User ID. This action is irreversible.

User-Controlled Privacy Permanent Deletion

What Will Be Deleted

  • Points and rewards balance
  • Daily task history and logs
  • Orders and redemption records
  • Points change logs
  • User cache and analytics identifiers

Once processed, deletion cannot be undone.

Exceptions

Some non-personal aggregated statistics (e.g., total anonymous usage counts) may be retained for analytics. These do not include identifiable data.

How to Request Deletion

  • In-app: Settings → Delete Account
  • Email: devcloper.url@gmail.com — include your ML User ID
